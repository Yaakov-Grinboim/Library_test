#  פרויקט סיכום מבואות 
#  ספרייה

המערכת שולטת בנתוני ספריה, 
יש בה מסד נתונים של לקוחות ומסד נתונים של ספרים,
המערכת שולטת בספריה ומאפשר לחפש ספרים או לקוחות, לעדכן להוסיף או למחוק לקוחות לרשימה או ממנה ולעדכן את מלאי הספרים הקיים להשאלה.
ממיין את הספרים לפי קטגוריות 
ובודק מי לקח איזה ספר
ומאפשר לעדכן לקיחת ספר והחזרתו.

המערכת שולחת את הנתונים ומעדכנת אותם בשרת באמצעות ספריית fastapi
ומעדכנת נתוני ריצה באמצעות logging לקובץ לוג מוגדר.

המערכת עובדת ע"י mysql ורצה על docker באמצעות הפקודה:

 "docker run --name my-mysql  -e MYSQL_ROOT_PASSWORD=secret  -e MYSQL_DATABASE=mydb  -p 3306:3306  -v mysql_data:/var/lib/mysql  -d mysql:latest"

:מבנה התיקיות

```
library-api/ 
│ 
│ 
├── main.py                 # הקובץ שמריץ את "תוכנת ניהול הספרייה"
├── database/               # תיקיית הנתונים הנשלטת באמצעות sql
│   ├── db_connection.py    # הפונקציות שיוצרות את מבנה הנתונים ואת הטבלאות
│   ├── book_db.py          # מכיל את הפקודות לעדכון מידע בטבלת הספרים
│   └── member_db.py        # מכיל את הפקודות לעדכון מידע בטבלת הלקוחות
├── routes/                 # תיקיית החיבור לשרת
│   ├── book_routes.py      # endpoints לשליחת וקבלת מידע על הספרים
│   ├── member_routes.py    # endpoints לשליחת וקבלת מידע על הלקוחות
│   └── report_routes.py 
├── logs/                   # תיקיית השליטה על החזרת המידע באמצעות לוגים 
│   └── app.log 
│ 
├── README.md 
├── requirements.txt 
└── .gitignore
```
